# Gaijinometer – Japan Tourism Heatmap

Interactive heatmap of the most visited tourist destinations across Japan, with date filtering by year, month, or custom range.

## Features

- Heatmap of ~40 popular spots across Japan (Tokyo, Kyoto, Osaka, Hokkaido, Okinawa, and more)
- Filter by year, month, year+month, or custom date range
- Top 10 rankings panel
- Spot detail with bar chart breakdown
- Visit data 2019–2026 (includes COVID impact)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React + Vite
- Leaflet + leaflet.heat
- OpenStreetMap tiles
